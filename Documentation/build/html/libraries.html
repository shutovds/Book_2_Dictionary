static analyze_book_words()[source]¶

This method provides words processing from dictionary. It download list of words for chosen book from database to the memory, and show it by frequency in descending order. User can process the words:

  • add word to Ignor or to other dictionary
  • remove word
  • change the word
  • pass the word

It is necessary for making your oun dictionary
